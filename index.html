1. Class Blueprint

A class is a blueprint for creating objects. It defines properties and behaviors that objects will use.

class Student {
    String name;
    int age;

    void show() {
        System.out.println(name + " " + age);
    }
}

2. Variables Data holders

Variables store values. Java has local variables, instance variables, and static variables.

class Demo {
    int x = 10;
    static int y = 20;

    void show() {
        int z = 30;
        System.out.println(x + y + z);
    }
}

3. Objects Instances

An object is a real instance of a class. It has its own values and can call the class methods.

Student s1 = new Student();
s1.name = "Abhishek";
s1.age = 20;
s1.show();

4. Methods Actions

Methods are blocks of code that perform tasks. They make code reusable and organized.

class Calc {
    int add(int a, int b) {
        return a + b;
    }
}

4a. Method Overloading Same name, different params

Overloading means using the same method name with different parameter lists.

class Calc {
    int add(int a, int b) { return a + b; }
    int add(int a, int b, int c) { return a + b + c; }
}

4b. Method Overriding Child version

Overriding happens when a child class defines its own version of a parent method.

class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}
class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("Bark");
    }
}

5. Inheritance Types in Java

Inheritance allows one class to acquire properties and methods of another class. Java supports single, multilevel, and hierarchical inheritance using classes. Multiple and hybrid inheritance are achieved using interfaces.

Single Inheritance

One child class inherits from one parent class.

class Animal {
    void eat() {
        System.out.println("Eating");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Barking");
    }
}

Multilevel Inheritance

One class inherits from a child class, forming a chain.

class GrandParent {
    void house() {
        System.out.println("Grandparent house");
    }
}

class Parent extends GrandParent {
    void bike() {
        System.out.println("Parent bike");
    }
}

class Child extends Parent {
    void mobile() {
        System.out.println("Child mobile");
    }
}

Hierarchical Inheritance

Multiple child classes inherit from the same parent class.

class Shape {
    void draw() {
        System.out.println("Drawing shape");
    }
}

class Circle extends Shape {
    void circleInfo() {
        System.out.println("Circle");
    }
}

class Square extends Shape {
    void squareInfo() {
        System.out.println("Square");
    }
}

Multiple Inheritance

Java uses interfaces to achieve multiple inheritance.

interface Printer {
    void print();
}

interface ScannerDevice {
    void scan();
}

class Machine implements Printer, ScannerDevice {
    public void print() {
        System.out.println("Printing");
    }

    public void scan() {
        System.out.println("Scanning");
    }
}

Hybrid Inheritance

Hybrid inheritance is a combination of two or more inheritance types. In Java, it is commonly done with interfaces.

interface A {
    void showA();
}

interface B extends A {
    void showB();
}

class C {
    void showC() {
        System.out.println("Class C");
    }
}

class D extends C implements B {
    public void showA() {
        System.out.println("A");
    }

    public void showB() {
        System.out.println("B");
    }
}

6. Functions Methods in Java

In Java, functions are called methods. They may return a value or be void.

public static int square(int n) {
    return n * n;
}

7. Arrays Fixed size

An array stores multiple values of the same type in one variable.

int[] nums = {10, 20, 30, 40};

for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]);
}

8. Vectors Dynamic list

Vector is a growable list from java.util. It automatically expands as items are added.

import java.util.Vector;

Vector v = new Vector<>();
v.add("Java");
v.add("Python");
System.out.println(v);

9. Strings Text

Strings store text and are immutable in Java, meaning they cannot be changed after creation.

String name = "Abhishek";
System.out.println(name.length());
System.out.println(name.toUpperCase());

10. Exception Handling Error control

Exception handling manages runtime errors using try, catch, finally, throw, and throws.

try {
    int a = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero");
} finally {
    System.out.println("Done");
}
